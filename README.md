# Curso Programación en Python — SII 2026 · Puerto Siracusa

> Curso de Python de 20 horas para funcionarios, **sin experiencia previa en
> programación**. Se aprende por laboratorios con narrativa continua: no estudias
> Python en abstracto, resuelves los encargos que te pone tu jefe.

## La historia

Eres **Analista de Datos** de la ficticia **Dirección de Rentas de Puerto
Siracusa**, un municipio costero. Tu jefe es **Don Arquímedes**, veterano de
Rentas, que te irá pasando tareas reales —triajes, cruces, informes— con las que
aprenderás Python paso a paso.

> «Dame una planilla limpia y un punto de apoyo, y moveré el presupuesto
> municipal.» — Don Arquímedes

## Dos rutas, un mismo destino

Cada lab se recorre de dos formas, y ambas terminan en el mismo verificador:

- 🛠️ **Artesano** — completas los `TODO` de las plantillas; escribes tú el código.
- 🔎 **Explorador** — ejecutas la solución, pero **prediciendo y modificando**
  de forma obligatoria; aprendes desarmando.

## Inicio rápido

1. Instala **`uv`** (gestor de entorno y de Python). Guía única macOS/Windows:
   **[`docs/setup-alumno.md`](docs/setup-alumno.md)**. No necesitas instalar
   Python: uv trae el 3.13 del curso.
2. Entra al primer laboratorio y sigue su `README`:
   ```bash
   cd labs/lab-01-primer-dia
   ```
